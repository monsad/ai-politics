**Script structure** (replicate seed_pageindex_konstytucja.py functions exactly, but batch):

```python
#!/usr/bin/env python3
"""
Batch seed PageIndex Cloud with the full ~50-doc corpus.
Extends scripts/seed_pageindex_konstytucja.py pattern for batch ingest.

Usage:
    python scripts/seed_pageindex_corpus.py [--dry-run] [--domain DOMAIN]
    
    --dry-run: Print URLs and source_refs, do not upload
    --domain DOMAIN: Only ingest docs for a specific domain

Output: Updates data/doc_manifest.json with {source_ref: {doc_id, title, domain, pages}}
Exit codes: 0=success, 1=partial failure (some docs failed), 2=config error
"""
```

Functions required:
1. `load_manifest(path) -> dict` — load existing doc_manifest.json (create empty dict if not exists)
2. `save_manifest(path, manifest) -> None` — write manifest atomically (write to .tmp, rename)
3. `find_existing_doc(client, api_key, source_ref) -> str | None` — same as Konstytucja script but checks metadata source_ref
4. `fetch_pdf(client, doc_entry, cache_dir) -> Path` — download to `data/{source_ref}.pdf` if not cached
5. `upload_pdf(client, api_key, pdf_path, doc_entry) -> str` — POST /doc, return doc_id
6. `wait_for_indexing(client, api_key, doc_id, timeout=600) -> None` — poll GET /doc/{doc_id}
7. `ingest_doc(client, api_key, doc_entry, manifest, cache_dir) -> str` — orchestrates 3-6, idempotent
8. `main()` — iterates CORE_DOCS + DOMAIN_DOCS, calls ingest_doc, saves manifest after each doc, prints summary

**Page budget enforcement:** Before starting each upload, print running page count from manifest. If estimate exceeds 950 pages (10% buffer before 1000 limit), prompt user to confirm or skip. Use `pages_hint` field from doc entry as advisory only — actual page count comes from PageIndex status response.

**Error handling:** If any doc fails with a 4xx/5xx, print error and continue to next doc (do not abort). At the end, print a summary of successes/failures. Exit code 1 if any failures, 0 if all succeeded.

**Cache directory:** `data/pdf_cache/` — not committed (add to .gitignore). The manifest is committed.
